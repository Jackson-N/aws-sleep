Amazon Alexa Sleep Game

Instructions

Click the button to tell Alexa when you'll go to bed, and she'll do the rest!



This project is a sleep timer and reminder for incentivising sleep. The user will prompt Alexa that they're going to bed, and will then be asked to know when they they're going to bed, and when they plan to wake up. Once those things are done, the user can rest assure knowing that Alexa will remind them when its time to wake up.

Once the time for waking up comes, Alexa will ask if they want to snooze or not. She will then do so based on the user's response. Once the user wants to wake up, Alexa will let the user know about how long they slept, including snooze time. The user gets points from sleeping consistantly, and can loose points based on times snoozed, or irragualr sleep times. The user is able to opt out of the game at any time, and can be used only as a sleep timer if the user so chooses.
